# DSA in Java

This repository documents my learning journey in **Data Structures & Algorithms (DSA)** using Java.
The goal is not only to understand theory, but also to apply each concept through **labs and small practical projects**.

---

## 📁 Structure

```bash
dsa-java/
│
├── material/        # course slides, notes, and references
├── weekly-labs/     # weekly lab implementations
├── projects/        # small practical projects for each topic
└── README.md
```

---

## 📚 Topics Covered

- OOP Review — classes, inheritance, polymorphism
- Algorithm Analysis & Recursion
- Collections & ArrayLists
- Linked Lists
- Stacks, Queues, Dequeues
- Trees & Binary Search Trees
- AVL Trees
- Priority Queues & Heaps
- Sets, Maps & Hash Tables
- Graphs — DFS, BFS, Dijkstra, Prim, Kruskal

---

## 🧪 Labs

Weekly implementations covering core logic and problem-solving for each topic.

---

## 🧩 Projects

Small projects that apply each concept to a real scenario:

| Concept | Project |
|---------|---------|
| Stack | Expression evaluator / undo system |
| Queue | Task scheduling simulation |
| Linked List | Dynamic data management |
| Trees | Hierarchical structure representation |
| Graphs | Shortest path and network optimization |

---

## 🎯 Objectives

- Build a strong understanding of DSA fundamentals
- Practice clean and efficient Java implementations
- Connect theory with real-world use cases
- Prepare for technical interviews
